# DATABASE SETUP
Instructions setup and add dummy data to your mongodb instance

1. startup your mongodb instance
2. open db_creation.js and enter the path to db_connection.js inside of 'require' on the first line
3. open db_connection.js and enter the mongodb url 
4. open mongosh in your terminal (open the terminal, type mongosh and enter)
5. enter this command: load("path to db_creation.js")
